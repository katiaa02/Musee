

# MuséeEvasion

Muséevasion est une plateforme dédiée aux amateurs de musées, offrant une expérience unique pour explorer et découvrir les trésors artistiques et culturels à travers le monde. Ce site captivant vous emmène dans un voyage virtuel, invitant les passionnés d'art, d'histoire et de culture à explorer une vaste collection de musées renommés, sans quitter le confort de chez eux.

## Prérequis
- Serveur local (par exemple, WAMP, XAMPP, ou tout autre serveur PHP/MySQL)
- Navigateur web moderne (Chrome, Firefox, Safari, etc.)


## API 

Le projet MuséeEvasion  exploite principalement deux API serveurs et une API client afin de fournir des informations sur les musées :

#### API Wikimedia
Cette API est utilisée pour obtenir des informations détaillées sur les musées, y compris leur localisation, leur histoire, leurs expositions actuelles et passées, ainsi que des images et des vidéos.

#### API Dataculture
Cette API fournit des données sur les événements culturels et artistiques se déroulant dans les musées. Elle permet aux utilisateurs de découvrir les expositions temporaires, les conférences, les ateliers et autres activités culturelles proposées par les musées partenaires.

#### API OPenStreetMap
Cette API fournit des fonctionnalités de cartographie et de géolocalisation pour aider les utilisateurs à visualiser et à naviguer vers les musées partenaires.

## Fonctionnalités Principales

Le site MuséeEvasion propose les fonctionnalités suivantes :

#### EXPLORER
Permet aux utilisateurs de découvrir les musées de France à travers des descriptions détaillées et des galeries photos. Ils peuvent explorer les différentes salles, les collections permanentes et les expositions temporaires.

#### RECHERCHE
Fonctionnalité de recherche permettant aux utilisateurs de trouver des musées en fonction de leur ville ou de leur nom. Cela facilite la recherche de musées spécifiques ou de musées situés à proximité.

#### INSCRIPTION/CONNEXION
Les utilisateurs peuvent créer un compte sur MuséeEvasion en fournissant leur adresse e-mail et en confirmant leur inscription par e-mail. Une fois inscrits, ils peuvent se connecter de manière sécurisée pour accéder à des fonctionnalités supplémentaires.

#### FAVORIS
Les utilisateurs peuvent sélectionner et ajouter des musées à leurs favoris. Cela leur permet de créer une expérience personnalisée en enregistrant leurs musées préférés et en y accédant facilement depuis leur profil.



## Technologies utilisées
